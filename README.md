# GDScript para Iniciantes

Este repositório contém anotações didáticas sobre GDScript e Godot,
escritas por um iniciante, para iniciantes.

O objetivo é explicar os conceitos de forma simples,
com exemplos práticos e linguagem acessível.

Documentação oficial utilizada como base:
https://docs.godotengine.org/en/stable/

Observação: este repositório está em desenvolvimento ativo por um único autor.
A adição de novos conteúdos ocorre de forma gradual, conforme disponibilidade.
O projeto será expandido continuamente.
